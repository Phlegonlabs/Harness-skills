# Harness Engineering Skills — Catalog

This repository is a multi-skill catalog. Each top-level directory ending in no `-prd` suffix and containing a `SKILL.md` is a published, installable skill.

## Published Skills

| Skill | Version | What it does | Install |
|---|---|---|---|
| `harness-engineering-orchestrator` | v1.7.0 | Turns an idea or existing repo into a repo-backed PRD-to-code delivery workflow with docs, runtime state, backlog, execution, and validation | `npx skills add https://github.com/Phlegonlabs/Harness-Engineering-skills --skill harness-engineering-orchestrator` |

## Adding a New Skill

See [docs/new-skill-guide.md](./docs/new-skill-guide.md) for the required directory structure, `SKILL.md` requirements, testing conventions, and PR checklist.
